Text Editor
===========

As a C++ software developer I need a good text editor. 

Key features:

- Performance
- Linux/Mac OS X
- Syntax Highlighting
- C++ code compilation
- Debugger
- Easy code navigation
- Work with git, hg and svn
- Terminal emulator
- Remote work via ssh
- Keyboard driven
- C++ plug-ins
- Tabs

Possible dependencies:

- C++ 11 language
- SDL
- OpenGL
- FreeType
